N64: Mupen64Plus 
---------------------
0  : Save state
1  : Increment State
2  : Load State
3  : Start button
4  : D-Pad up 
5  : D-Pad right
6  : D-Pad down
7  : D-Pad left
8  : Z trigger
9  : Fast Forward
10 : L bumper
11 : R bumper
12 : C button Up
13 : C button Down
14 : A button
15 : B button
16 : Exit Emulator
X1-: Analog left
X1+: Analog right
Y1-: Analog up
Y1+: Analog down
X2-: C button left
X2+: C button right
Y2-: C button up
Y2+: C button down
