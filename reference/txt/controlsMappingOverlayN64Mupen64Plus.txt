N64: Mupen64Plus 
---------------------
0  : Next state
1  : Volume-
2  : Volume+
3  : Start button
4  : D-Pad up 
5  : D-Pad right
6  : D-Pad down
7  : D-Pad left
8  : Z trigger
9  : Fast Forward
10 : L bumper
11 : R bumper
12 : Save state
13 : Load state
14 : A button
15 : B button
16 : Exit Emulator
X1-: Analog left
X1+: Analog right
Y1-: Analog up
Y1+: Analog down
X2-: C button left
X2+: C button right
Y2-: C button up
Y2+: C button down
